# Platformer Game

A simple 2D platformer game built with HTML5, JavaScript, and Kaboom.js using basic geometric shapes instead of sprites.

## Features

- **Player Movement**: Move left/right with arrow keys
- **Jumping**: Jump with spacebar
- **Platforms**: Multiple floating platforms to jump on
- **Obstacles**: Spikes that cause game over
- **Collectibles**: Golden coins to increase score
- **Physics**: Basic gravity and collision detection
- **Game Over**: Restart functionality when you lose

## How to Run

1. Simply open `index.html` in a modern web browser
2. No additional setup or server required
3. The game will start automatically

## Controls

- **← → Arrow Keys**: Move left/right
- **Spacebar**: Jump
- **R**: Restart game (when game over)

## Game Mechanics

- The player (blue square) can move left and right
- Jump from platform to platform
- Collect golden coins to increase your score
- Avoid red spikes - they cause instant game over
- If you fall off the platforms, it's game over
- Press R to restart when you lose

## Technical Details

- Built with Kaboom.js game framework
- Uses HTML5 Canvas for rendering
- Simple collision detection system
- Basic physics with gravity and jumping
- Responsive design with centered game container
- No external dependencies beyond Kaboom.js CDN

## File Structure

- `index.html` - Main HTML file with styling
- `game.js` - Game logic and mechanics
- `README.md` - This documentation file

Enjoy the game!
